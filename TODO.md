## On Linux (Github)
- Edit the usernames in the ```datasets/usernames.txt```
- Run: ```chmod +x linux_setup.sh;./linux_setup.sh```
- Don't know how to use it then i recommend you to follow the instructions here: **[README](README.md)**

## On Linux (PyPI)
- Run: ```python3 -m venv myvenv;source myvenv/bin/activate;pip install personacipher```
- Edit the usernames in the ```nano ~/venv/lib/python3.13/site-packages/persona_cipher/datasets/usernames.txt```
- For installation via PyPI, please refer to the official documentation here: **[README](https://pypi.org/project/PersonaCipher/)**
## Note
**The program allows only authorized Google searches and is designed to search for famous individuals. This restriction is in place to respect privacy and ensure that the system is used ethically.**
